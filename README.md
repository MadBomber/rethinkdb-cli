# Rethinkdb Command Line Interface

WIP: A cli for rethinkdb written in ruby. Stay tuned.

At moment there a very simply version, to try:

1. Install the gem
2. Run `rethinkdb-cli`
3. Use `$c` variable to send requests to Rethinkdb server

# Rethinkdb Command Line Interface

A cli for rethinkdb written in ruby.

At moment there a very simply version, to try:

1. Install the gem `gem install rethinkdb-cli`
2. Run `rethinkdb-cli`
3. Use `r` to access rethinkdb database
